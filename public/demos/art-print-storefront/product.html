Shopify Code Panel -- Product Page 
Write a Shopify OS 2.0 Liquid section for a fine-art print
product page.
Requirements:
- Section name: "Print Product Detail"
- Editorial image using product.featured_image
- Print size variant selector (iterate product.variants)
- Edition metafields: namespace "print_details", keys
    edition_type, edition_size, edition_number
- Add to cart form targeting the selected variant
- Output the complete product.liquid file, no dependencies
  1. 1 Metafield access: use product.metafields.print_details.edition_type.value (OS 2.0 dot notation)
  2. 2 Variant loop: iterate product.variants to build size buttons, not product.options
  3. 3 Add to cart: form needs action="/cart/add", hidden name="id" input set to {{ variant.id }}
  4. 4 Image loading: product.featured_image | image_url: width: 1200 | image_tag: loading: "lazy"
  5. 5 JSON-LD: include product schema markup for SEO -- Claude often forgets this step
  1. 1 Using bracket syntax product.metafields["print_details"]["edition_type"] -- Liquid requires dot notation only
  2. 2 Forgetting the {% form 'product', product %} wrapper -- the Add to Cart button will silently fail without it
  3. 3 Reading variant.price in JavaScript without dividing by 100 -- Shopify returns prices in cents (e.g., 18000 = $180)
