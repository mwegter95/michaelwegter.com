Write a Shopify OS 2.0 Liquid section for a horizontal-scrolling print gallery.
Requirements:
- Section name: "Horizontal Gallery"
- Each gallery item is a block of type "print" with settings:
    image_picker, text (title), text (base_price)
- CSS: display:grid, grid-auto-flow:column, overflow-x:auto,
    scroll-snap-type:x mandatory
- JS: drag-to-scroll with mousedown/mousemove tracking,
    prev/next arrow buttons
- Add {% schema %} with max_blocks: 20 and a preset
    named "Horizontal Gallery"
- Output the complete .liquid file, no external dependencies
  1. 1 Schema: confirm "type": "image_picker" -- NOT "type": "image" (common LLM error)
  2. 2 Block attributes: every block div needs {{ block.shopify_attributes }} for theme editor targeting
  3. 3 Image filter: use | image_url: width: 800 | image_tag (OS 2.0 syntax), not the deprecated | img_url
  4. 4 Preset: the presets array must exist for the section to appear in the "Add section" panel
  5. 5 Mobile: test overflow-x: auto on Safari iOS -- may need -webkit-overflow-scrolling: touch

  2. 2 Missing {{ block.shopify_attributes }} on block wrapper divs -- makes blocks un-selectable in the theme editor
  3. 3 Placing {% schema %} inside an {% if %} block -- schema must be top-level and appear exactly once per section file
